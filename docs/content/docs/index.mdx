# TezX v4 (Bun Edition)

**TezX** is a modern, ultra-lightweight, and high-performance JavaScript framework built **specifically for Bun**.
It provides a **minimal yet powerful API**, seamless environment management, and a high-concurrency HTTP engine for building fast, scalable web applications.

---

## Key Features (Bun-only)

* ‚ö° **Ultra-Fast Performance** ‚Äì Optimized for Bun‚Äôs native HTTP engine and streams.
* üß© **Minimal & Intuitive API** ‚Äì Clean and simple routing, middleware, and static file serving.
* üîå **Flexible Middleware** ‚Äì Stackable, composable, and easy to manage.
* üóÇ **Static File Serving** ‚Äì Efficient streaming, automatic MIME detection, optional **ETag** support.
* üß≠ **Dynamic Routing** ‚Äì Pattern-based, parameterized, and nested routes.
* üåç **Environment-First Design** ‚Äì Easy `.env` loading with typed access via `Bun.env`.
* üîê **Secure by Default** ‚Äì Safe headers, CORS, caching, and other best practices baked in.

## Installation


## Environment Setup

TezX v4 is **Bun-native** and leverages `Bun.env` for configuration:

```ts
import { TezX } from "tezx";

const app = new TezX();
const port = Number(Bun.env.PORT || 3000);
```

---

## Quick Start

```ts
import { TezX } from "tezx";
import { serveStatic } from "tezx/static";

const app = new TezX();

// ‚úÖ Simple GET route
app.get("/", (ctx) => ctx.html("<h1>Welcome to TezX v4 for Bun</h1>"));

// ‚úÖ Static files
app.static(serveStatic("/public", "./public"));

// ‚úÖ Middleware example
app.use((ctx, next) => {
  console.log(`Incoming request: ${ctx.req.url}`);
  return next();
});

// ‚úÖ Start server
Bun.serve({
  port,
  fetch(req, server) {
    return app.serve(req, server);
  },
});

console.log(`üöÄ TezX v4 running at http://localhost:${port}`);
```

## üóÇ Static File Serving

```ts

```

* Efficient **streaming** for large files.
* Automatic **MIME type detection**.
* Optional **ETag** and caching for faster reloads.

## Routing Examples

```ts
// GET request
app.get("/about", (ctx) => ctx.html("<h1>About TezX</h1>"));

// POST request
app.post("/submit", (ctx) =>
  ctx.json({ message: "Form submitted successfully" })
);
```

## Error Handling

```ts
app.onError((err, ctx) => {
  console.error(err);
  return ctx.status(500).json({ error: "Internal Server Error" });
});
```
